通过 GitHub PR 投稿插件

插件、支付接口、模板全部托管在公开仓库 NoDoorAction/Acg-Faka-Plugins ， 想发布新插件只需提交一个 Pull Request — 仓库管理员审核合并后，所有用户的"应用商店"页就会立刻看到。
Fork 仓库
在 GitHub 上 fork NoDoorAction/Acg-Faka-Plugins 到你的账号下，clone 到本地。
添加插件目录
在 plugins/、pays/ 或 themes/ 下新建一个目录， 放进 Config/Info.php、入口文件、可选的 icon.png / install.sql / README.md。
更新索引 + 提 PR
运行 node scripts/gen-index.js 重生成 plugins.json，然后 commit + push + 提 PR。
安全提示：插件代码会运行在用户服务器上，仓库管理员合并 PR 前会进行人工审核。 如果你只是想在自己服务器上跑一个临时插件，不需要发 PR —— 在"应用商店"页点"本地 zip 安装"， 上传打包好的 zip 即可。
打开插件仓库 查看投稿规范 发起 Pull Request
